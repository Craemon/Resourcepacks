Short instruction on how to use this:

1.Download and install the "Better Chroma Key Mod"
2.Download this pack and extract it!
3.Change out the texture under "assets/betterchromakey/textures/blockreplace_me.png" to your desired texture!
4.Recompress the files of the pack to a .zip
5.Load the pack in Minecraft

Done, For help please join the Discord server and ask in the appropriate channel!
